# AgroNexus - Smart Shepherd UI

A premium React Native mobile application UI for livestock monitoring.

## Features
- **Map View**: Real-time GPS tracking with geofencing and animal status dots.
- **AI Vet**: Automated diagnostic insights and behavior analysis charts.
- **Herd Management**: Searchable list of all livestock with health status.
- **Rich Alerts**: Detailed notification system for critical events.
- **Premium Design**: Dark orange branding, glassmorphism elements, and smooth typography.

## Design System
- **Primary Color**: #FF6B00 (Agro Orange)
- **Background**: #FFFFFF (Main) / #FFF7ED (Soft tint)
- **Typography**: System (Standard / Medium / Bold)
- **Border Radius**: 12px (Cards) / 20px (Interactive buttons)

## Tech Stack
- Expo / React Native
- React Navigation (Stack & Bottom Tabs)
- Lucide React Native (Icons)
